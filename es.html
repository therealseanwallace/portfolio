weatherPlusPlus

Una aplicación del tiempo fullstack (MERN). Los usuarios pueden consultar el tiempo para una ubicación dada. La aplicación React consulta la API RESTful del backend, que a su vez verifica si el tiempo para esa ubicación ya está almacenado en la base de datos MongoDB. Si no es así, entonces el backend consulta la API de OpenWeatherMap y guarda los resultados en la base de datos antes de devolver la respuesta al frontend.
