Calculadora Cuántica de Venta Engañosa de Energía

He entregado, de principio a fin, una aplicación completa y propia para un cliente que ayuda a los profesionales legales con sus reclamaciones relacionadas con la venta engañosa de energía empresarial. La aplicación utiliza una base de datos PostgreSQL para trabajar con alrededor de 7 millones de filas hasta ahora. Cuenta con:

  • Funcionalidad CRUD completa
  • Diseño responsive
  • Autenticación y autorización stateless usando Passport y JWTs
  • API RESTful construida con Express
  • Control de acceso por rol
  • Generación de PDFs de certificados de las búsquedas de los usuarios
  • Almacenamiento de archivos PDF en Google Cloud Storage para su posterior recuperación
  • Error handling personalizado
  • Middleware personalizado
  • Validación personalizada
  • Paginación personalizada
  • Cobertura de pruebas exhaustiva de funciones vitales de la aplicación
  • Base de datos PostgreSQL
  • Frontend y proxy de API Next.JS
  • Alojamiento API y base de datos en Heroku

justEducationTenders

Una aplicación fullstack que rastrea, almacena y muestra licitaciones de educación del Reino Unido en tiempo casi real. Construida en Next.JS con un servicio crawler en una VM separada. Hosted usando Vercel, AWS, Cloudflare y GitHub Pages.
He crecido mucho como desarollador desde que hice este proyecto, y haría muchas cosas diferentes si lo volviera a hacer. Pero me enseñó mucho sobre Next.JS y la resolución de problemas en general.
