Energy Misselling Quantum Calculator

I delivered, from end-to-end, a fullstack, proprietary app for a client which assists legal practicioners with their claims relating to business energy misselling. The app uses a PostgreSQL database to work with about 7 million rows so far. It features:

  • Full CRUD functionality
  • Responsive design
  • Stateless authentication and authorisation using Passport and JWTs
  • RESTful API built with Express
  • Role-based access control
  • Generation of certificate PDFs for user's searches.
  • Google Cloud Storage of PDF files for later retrieval.
  • Custom error handling
  • Custom middleware
  • Custom validation
  • Custom pagination
  • Comprehensive test coverage of vital app functions
  • PostgreSQL database
  • Next.JS frontend and API proxy
  • Heroku API and database hosting
This was a proprietary work to which all rights belong to the client. I am unable to provide access to the codebase.

justEducationTenders

A full stack app which crawls, stores, and displays UK education tenders in near-real time. Built in Next.JS for both front and backend with a crawler service running on a separate VM. Hosted using Vercel, AWS, Cloudflare, and GitHub Pages. I've grown a lot as a developer since making this one, and I'd do a lot of things differently if I made it again. But it did teach me a lot about Next.JS and problem-solving in general.
